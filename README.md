#  Shadow Insure Bot

**Shadow Insure Bot** is a simple, interactive chatbot designed to help users with basic insurance-related queries. Built using **HTML**, **Tailwind CSS**, and **JavaScript**, this lightweight bot supports both **text** and **voice** interactions for an accessible and friendly experience.

---

##  Features

-  **Clean & Modern UI**: Sleek and user-friendly interface powered by Tailwind CSS.
- **Voice Input (Speech Recognition)**: Speak directly to the bot using your deviceâ€™s microphone.
-  **Voice Output (Text-to-Speech)**: Bot reads out responses for a conversational feel.
-  **Real-Time Bot Status**: See real-time indicators like _Online_, _Listening_, and _Thinking..._.
-  **Insurance FAQs**: Pre-loaded answers for topics like:
  - Policies
  - Claims
  - Health Insurance
  - Life Insurance
-  **Responsive Design**: Optimized for mobile, tablet, and desktop views.
